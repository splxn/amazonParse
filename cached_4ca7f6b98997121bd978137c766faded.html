4.0 out of 5 starsOutstanding value for an ultra book if you can put in some time to fix some issues
By DW on January 25, 2016
Verified Purchase
I was about to pull the trigger on purchasing a much older and refurished Lenovo Thinkpad when I thankfully stumbled upon this new model.

First, how to make it much better:

Fix the unaccepable wifi problem that is being reported by many owners (more on this below). This is the reason I only rated @ 4 stars. With this fixed, it's a solid 4.5 out of 5 for the money and an amazing value.
There is a huge Windows update that seemed to make a copy of the Windows directory as part of the process. You can remove this later using the Windows disk cleanup feature (Google it, it's not that difficult). This freed up about 18.5 GB of storage, which is a big deal considering that this SSD is a bit small at 120 GB.
Install all of your essential (and some of the best ) free apps from ninite.com. This will save you a TON of time while also ensuring that your apps are installed without any bloat or malicious code.
Remove the pre-installed antivirus software. In this case it is McAfee, but you should choose a better solution from ninite.com.
Run Windows Updates until is is done updating. Reboot as instructed.

All in all, I would say that this is an outstanding value for the money and you could do worse than buying as many as you can.

The good:

The laptop looks and feels more premium than I would have expected for a laptop in this price range.
The battery should last up to 9 hours when used for light tasks like writing papers, etc.
Surprisingly decent screen with a bright and crisp display
Decent performing SSD drive with ample storage (120GB).
Decent amount of RAM (4GB) with the option to install up to 8GB easily by yourself and for about $40.Read more ›
