5.0 out of 5 starsGreat laptop, fast. I recommend.
By Meowmix123 on December 29, 2015
I purchased this laptop from another website.*

Although I've only had it for about 2 weeks, I love this laptop. I bought it to replace my 5 yr old Toshiba. This is my 1st time using Windows 10 and it is very easy to learn. It has movable tiles and apps in the start bar. It's very user friendly. So far I do not like the new Internet browser as it was having some problems. But simple fix is to download Firefox which is very fast. The keyboard is easy to adjust to ( every new keyboard is a little weird to me at 1st especially after always being on a smartphone ). I like that it has volume control keys. I wouldn't call this a "gaming laptop" but I do use it as one!! I can play the new world of warcraft without any problems or lagging! Yay! The Webcam is okay, not too spectacular or anything but not bad. I really only have one complaint and that is the mouse. It is soooo sensitive. Which is good... but bad. A single cat hair on the mouse pad will make the mouse freeze or not work properly. Any small amount of moisture will also do the same so beware. Other that that it's an amazing laptop. Tons of storage and it's so fast! Web pages load almost instantly with no waiting or lags. Nothing crashes so far. Anyone with any computer knowledge would know 1t of memory and 8g of ram for this price is an awesome deal! I'm very pleased with this purchase and highly recommend.
